• • 에디터에서 작성한 내용을 가져 오거나(GetHtmlContents), 에디터에 작성된 내용을 set(SetHtmlContents)할 수 있습니다.
  • • GetHtmlContents API는 아래와 같이 get 형태와 callback 함수를 지정해 주시면 됩니다.
    예: GetHtmlContents({type: 'htmlex', callback: function (paramObj) { alert(paramObj.strData); }}, editorId)
  • • SetHtmlContents API는 에디터에 넣을 html 소스를 string으로 첫번째 파라미터로 넣고 호출하시면 됩니다.
