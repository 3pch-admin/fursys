K Editor :: SystemTitle

XML : <system_title>.....</system_title>, Script : SystemTitle

Script 설정이 우선 적용됩니다.
다이얼로그창 상단 문구 변경 시 설정합니다.

remarks

기본값은 "K Editor" 이고, "0" 으로 설정 시 사용하지 않음, 그 이외의 값을 입력하시면 해당 값으로 으로 표시 됩니다.

sample code

다이얼로그창 상단 문구를 Custom Title 로 설정합니다.

XML

.....
<setting>
    .....
    <system_title>Custom Title</system_title>
    .....
</setting>
.....

Script

<script type="text/javascript" src="raonkeditor/js/raonkeditor.js"></script>	
     
<!-- ..... 생략 ..... -->
     
<div style="width:900px;height:550px">  
    <script type="text/javascript">
        var raonkParam = {
            .....
            SystemTitle: 'Custom Title'
        }

        new RAONKEditor(raonkParam);
    </script>
</div>
