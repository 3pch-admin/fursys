K Editor :: AddUserCssUrl

void AddUserCssUrl(url, editorName)

에디터에 사용자의 css를 적용합니다.

return value

없음.

parameters

url  사용자의 css url 주소 입니다.
editorName  접근할 에디터의 id 입니다.

remarks

url은 http://를 포함한 전체 경로를 입력해야 합니다.

sample code

<!-- ..... 생략 ..... -->

<script type="text/javascript" src="raonkeditor/js/raonkeditor.js"></script>	
<script type="text/javascript">

    function RAONKEDITOR_CreationComplete(){
        // 에디터가 로드된 후 사용자의 css를 에디터에 적용시킵니다.
        var url = "http://www.raonk.com/css/user.css";
        RAONKEDITOR.AddUserCssUrl(url, "editor1");
    }

</script>	

<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">  
    <!-- 에디터 생성 -->  

    <script type="text/javascript">
         new RAONKEditor({Id:'editor1'});
    </script>

</div>
