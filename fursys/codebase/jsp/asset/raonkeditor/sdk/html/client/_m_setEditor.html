K Editor :: SetEditor

void SetEditor(editor_id)

다중 에디터 사용시 접근할 에디터를 설정합니다.
다중 에디터를 생성하여 작업할 경우 유용하게 쓰일 수 있습니다.

return value

없음.

parameters

editor_id  접근할 에디터의 id를 의미합니다.

remarks

없음.

sample code

<!-- ..... 생략 ..... -->

<script type="text/javascript" src="raonkeditor/js/raonkeditor.js"></script>	
<script type="text/javascript">

    function raonk_editor_loaded_event() {            
        // 접근하려는 에디터를 editor2로 설정합니다.
        RAONKEDITOR.SetEditor('editor2');
        // parameter를 넣지 않아도 editor2의 body 소스를 가져올 수 있습니다.
        RAONKEDITOR.getBodyValue();

        // 접근하려는 에디터를 editor1로 설정합니다. -->
        RAONKEDITOR.SetEditor('editor1');
        // parameter를 넣지 않아도 editor1의 웹 접근성 단계를 2단계로 설정할 수 있습니다.
        RAONKEDITOR.setAccessibility('2');
    }

</script>
