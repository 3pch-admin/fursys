K Editor :: ImageAutoFit

XML : <image_auto_fit>.....</image_auto_fit>, Script : ImageAutoFit

Script 설정이 우선 적용됩니다.
이미지파일 업로드시 에디터 화면 너비에 맞게 표시할지 여부를 설정합니다.

remarks

기본값은 "0" 이며 원본 크기대로 화면에 표시하고, "1"로 설정하면 에디터 화면 너비에 맞게 표시합니다.
"1" 로 설정하는 경우, 실제 이미지 사이즈가 변경되지는 않습니다.

sample code

이미지파일 업로드시 에디터 화면 너비에 맞게 표시하도록 설정합니다.

XML

.....
<uploader_setting>
    .....
    <image_auto_fit>1</image_auto_fit>
    .....
</uploader_setting>
.....

Script

<script type="text/javascript" src="raonkeditor/js/raonkeditor.js"></script>	
 
<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">   
    <script type="text/javascript">

            .....

        }

    </script>
</div>
