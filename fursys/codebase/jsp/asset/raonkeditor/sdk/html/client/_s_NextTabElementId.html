K Editor :: NextTabElementId

Script : NextTabElementId

Tab키를 입력하면 마지막 포커스가 설정한 아이디로 이동 합니다.
(단, 포커스를 줄 객체 아이디를 설정 시)

remarks

필요시 NextTabElementId: '아이디값' 으로 설정합니다.

sample code


<script type="text/javascript" src="raonkeditor/js/raonkeditor.js"></script>	
 
<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">  
    <script type="text/javascript">

        // 마지막 포커스가 설정한 ID값 raonk 으로 이동합니다.
        var raonkParam = {
            .....
            NextTabElementId: 'raonk'
        }
        new RAONKEditor(raonkParam);
    </script>       
</div>
