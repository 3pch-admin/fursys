K Editor :: AdjustCursorInTable

XML : <table_property adjust_cursor_in_table="....."></table_property>
Script : AdjustCursorInTable

Script 설정이 우선 적용됩니다.
크롬, 오페라, 사파리 브라우저에서는 테이블 셀간 방향키 이동이 지원되지 않습니다.
에디터에서 자체 처리된 크롬, 오페라, 사파리 브라우저에서의 방향키로 셀간 커서 이동 기능 사용여부를 설정합니다.

remarks

기본값은 "1"로 크롬, 오페라, 사파리 브라우저에서의 방향키로 셀간 커서 이동 기능 사용이고,
"0" 으로 설정 시 크롬, 오페라, 사파리 브라우저에서의 방향키로 셀간 커서 이동 기능을 사용하지 않습니다.

sample code

크롬, 오페라, 사파리 브라우저에서의 방향키로 셀간 커서 이동 기능 사용여부를 사용으로 설정합니다.

    .....
    <table_property adjust_cursor_in_table="1">
        .....
    </table_property>
    .....
