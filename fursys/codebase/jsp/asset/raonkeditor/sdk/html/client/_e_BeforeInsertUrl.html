K Editor :: RAONKEDITOR_BeforeInsertUrl

RAONKEDITOR_BeforeInsertUrl(editorId, paramObj)

에디터에 파일 업로드 후 url 삽입 전에 발생합니다.

return value

파라미터로 넘어온 url을 수정하여 리턴합니다.

parameters

editorId  에디터의 id입니다.
paramObj.strUrl  업로드된 파일의 웹경로입니다.

remarks

이벤트에서 받은 url을 다른 파일의 url로 변경하여 리턴하면 변경한 파일이 에디터에 표현됩니다.

sample code

<!-- ..... 생략 ..... -->

<script type="text/javascript" src="raonkeditor/js/raonkeditor.js"></script>
<script type="text/javascript">

    function RAONKEDITOR_BeforeInsertUrl(editorId, url) {
        // 에디터에 파일 업로드 후 해당 이벤트가 호출됩니다. 
        // 이 부분에서 url을 가공 처리 하신 후 리턴 합니다.

        return url;
    }

</script>
