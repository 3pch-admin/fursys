K Editor :: FocusInitObjId

Script : FocusInitObjId

에디터 로드 후 설정한 아이디를 가진 객체에 포커스를 설정합니다.

remarks

필요시 FocusInitObjId: '아이디값' 으로 설정합니다.

sample code

<script type="text/javascript" src="raonkeditor/js/raonkeditor.js"></script>	
 
<!-- ..... 생략 ..... -->
 
<div border="0" style="width:900px;height:550px">    
    <script type="text/javascript">
            
        var paramObj = {
            Id: 'editor1',          
            // 에디터 포커스를 title 아이디로 설정합니다.
            FocusInitObjId: 'title'  
        };
        
        new RAONKEditor(paramObj);

    </script>       
</div>
