K Editor :: ShowDialogPosition

XML : <showdialog_pos>.....</showdialog_pos>, Script : ShowDialogPosition

Script 설정이 우선 적용됩니다.
에디터 다이얼로그창이 표시될 위치를 설정합니다.

remarks

기본값은 "1" 이고, 에디터 영역 중앙에 표시됩니다.
이외의 값이면 브라우저 중앙에 표시 됩니다.

sample code

에디터 다이얼로그창 위치를 브라우저 중앙에 설정합니다.

XML

.....
<setting>
    .....
    <showdialog_pos>100</showdialog_pos>
    .....
</setting>
.....

Script

<script type="text/javascript" src="raonkeditor/js/raonkeditor.js"></script>	
 
<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">  
    <script type="text/javascript">
        var raonkParam = {
            .....
            UserMessage: {
                ShowDialogPosition: '100'
            }
        }

        new RAONKEditor(raonkParam);
    </script>
</div>
