K Editor :: SetDefaultSavePath

string getToSavePathUrl(editorName)

에디터에 설정된 ToSavePathUrl(서버에 연결 된 저장소의 URL 경로) 값을 가져옵니다.

return value

ToSavePathUrl로 설정된 값을 리턴합니다.

parameters

editorName  ToSavePathUrl 값을 가져올 에디터의 id를 의미합니다.

remarks

에디터를 1개만 생성했을 경우 parameter를 입력하지 않아도 ToSavePathUrl을 가져올 수 있습니다.
예) id가 editor1인 에디터를 생성했을 경우
     RAONKEDITOR.getToSavePathUrl('editor1'), RAONKEDITOR.getToSavePathUrl()  모두 editor1 에디터의 ToSavePathUrl 값을 가져옵니다.

에디터를 2개 이상 생성했을 경우 parameter를 반드시 입력해야 합니다.
예) id가 editor1, editor2인 에디터를 생성했을 경우
     RAONKEDITOR.getToSavePathUrl('editor1')  editor1 에디터의 ToSavePathUrl 값을 가져옵니다.
     RAONKEDITOR.getToSavePathUrl('editor2')  editor2 에디터의 ToSavePathUrl 값을 가져옵니다.
