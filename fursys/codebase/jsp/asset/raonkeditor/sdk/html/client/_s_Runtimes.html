K Editor :: Runtimes

XML : <runtimes>.....</runtimes>, Script : Runtimes

Script 설정이 우선 적용됩니다.
agent 설정시 agent 모드가 적용됩니다.
html5 설정시 html5 모드가 적용되며, html5모드는 ie10 이상만 지원합니다
(html5가 지원되지 않는 브라우저로 접속을 하게 되면 에디터에서 자동으로 agent모드로 구동됩니다.) 예) "agent" or "html5"

remarks

agent 설정(기본값)시 모든 브라우저가 Agent모드로 동작됩니다.

html5 설정시 Html5가 지원되는 브라우저는 Html5모드로,
지원되지 않는 브라우저는 Agent모드로 동작됩니다.

sample code

에디터를 Agent 모드로 설정합니다.

XML

.....
<setting>
    .....
    <runtimes>agent</runtimes>
    .....
</setting>
.....

Script

<script type="text/javascript" src="raonkeditor/js/raonkeditor.js"></script>
