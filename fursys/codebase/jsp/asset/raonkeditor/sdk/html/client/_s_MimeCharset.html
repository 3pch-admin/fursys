K Editor :: MimeCharset

XML : <mime_use mimeCharset="....."></mime_use>, Script : MimeCharset

MIME에 사용할 문자 인코딩(character encoding) 형태를 설정합니다.

remarks

기본값은 "utf-8" 입니다.

sample code

인코딩 형태를 euc-kr 로 설정합니다.

XML

.....
<license>
    .....
    <mime_use mimeCharset="euc-kr"></mime_use>
    .....
</license>
.....

Script

<script type="text/javascript" src="raonkeditor/js/raonkeditor.js"></script>	
 
<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">  
    <script type="text/javascript">

        var raonkParam = {
            .....
            MimeCharset: 'euc-kr'
        }

        new RAONKEditor(raonkParam);
    </script>
</div>
