K Editor :: MimeRemoveHeader

Script : MimeLocalOnly

MIME 인코딩 생성 중에 "Subject", "DataData" 및 "X-Generator" 값을 생성하지 않습니다.

remarks

기본값은 "0"으로 헤더를 기록하여 MIME 인코딩 생성, "1"로 설정 시 헤더를 제거하고 MIME 인코딩을 생성합니다.

sample code

<script type="text/javascript" src="raonkeditor/js/raonkeditor.js"></script>	
 
<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">  
    <script type="text/javascript">

        var raonkParam = {
            .....
            // 헤더를 제거하고 MIME 인코딩을 생성합니다.
            MimeRemoveHeader: '1'
        }

        new RAONKEditor(raonkParam);
    </script>
</div>
