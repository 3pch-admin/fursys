K Editor :: ShowToolbar

void ShowToolbar(value, editor_id)

에디터의 툴바는 상단, 하단 두 개로 구성되어 있습니다.
이 설정은 상단의 툴바를 표시하거나 숨기는 기능을 설정합니다.

return value

없음.

parameters

value  툴바에 대한 설정값을 의미합니다.
editor_id  에디터의 id를 의미합니다.

remarks

사용 용도에 따라서 상단 툴바를 표시하지 않기를 원할 때 사용하면 유용 할 수 있습니다.
“0” 이면 툴바를 모두 숨기고,
“1” 이면 상단 툴바를 표시합니다.
“2” 이면 하단 툴바를 표시합니다.
“3” 이면 상단, 하단 툴바를 모두 표시합니다.
