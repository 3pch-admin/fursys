K Editor :: SetSize

void SetSize(new_width, new_height, editorName)

에디터 로드 후 에디터의 크기를 정해줍니다.

return value

없음.

parameters

new_width  에디터의 가로 값을 의미합니다.
new_height  에디터의 세로 값을 의미합니다.
editor_id  설정할 에디터의 id를 의미합니다.

remarks

에디터를 1개만 생성했을 경우 editor_id를 입력하지 않아도 에디터 크기를 설정할 수 있습니다.
예) id가 editor1인 에디터를 생성했을 경우
     RAONKEDITOR.SetSize("700", "500", "editor1"), RAONKEDITOR.SetSize("700", "500")  모두 editor1 에디터의
     크기를 설정합니다.

에디터를 2개 이상 생성했을 경우 editor_id를 반드시 입력해야 합니다.
예) id가 editor1, editor2인 에디터를 생성했을 경우
     RAONKEDITOR.SetSize("700", "500", "editor1")   editor1 에디터의 크기를 설정합니다.
     RAONKEDITOR.SetSize("700", "500", "editor2")   editor2 에디터의 크기를 설정합니다.
