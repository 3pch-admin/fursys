K Editor :: UserFieldID,

K Editor :: UserFieldValue

Script : UserFieldID, Script : UserFieldValue

파일 업로드에 사용되는 User 태그의 name과 value를 설정합니다.

remarks

User태그의 name은 UserFieldID로 설정합니다.
기본값은 "Userdata" 로 설정됩니다.

User태그의 value는 UserFieldValue로 설정합니다.
기본값은 빈값으로 필요 시 UserFieldValue: "newValue"로 설정합니다.

sample code

<script type="text/javascript" src="raonkeditor/js/raonkeditor.js"></script>	
 
<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">  
        <script type="text/javascript">
            
        var raonkParam = {
            .....
            // 에디터 User 태그의 name을 raonkID로 변경합니다.
            UserFieldID: 'raonkID'
            // 에디터 User 태그의 value를 raonkVal로 변경합니다.
            UserFieldValue: 'raonkVal'
        }

        new RAONKEditor(raonkParam);

        </script>       
</div>
