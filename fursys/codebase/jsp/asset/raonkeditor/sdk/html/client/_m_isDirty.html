K Editor :: IsDirty

bool IsDirty(editor_id)

에디터 내용이 변경 되었는지 확인 할 수 있는 설정 입니다.
(이 함수를 사용하시려면 Editor에 SetDirty함수를 적용한 시점부터 사용해야 합니다.)

return value

내용 변경 사항이 있으면 true, 없으면 false를 리턴 합니다.

parameters

editor_id  내용 변경 사항 유무를 확인할 에디터의 id를 의미합니다.

remarks

에디터를 1개만 생성했을 경우 parameter를 입력하지 않아도 내용 변경 사항 유무를 확인 할 수 있습니다.
예) id가 editor1인 에디터를 생성했을 경우
     RAONKEDITOR.IsDirty("editor1"), RAONKEDITOR.IsDirty()  모두 editor1 에디터 디자인 영역에 작성된 내용 변경 사항을 확인합니다.

에디터를 2개 이상 생성했을 경우 parameter를 반드시 입력해야 합니다.
예) id가 editor1, editor2인 에디터를 생성했을 경우
     RAONKEDITOR.IsDirty("editor1")  editor1 에디터 디자인 영역에 작성된 내용 변경 사항을 확인합니다.
     RAONKEDITOR.IsDirty("editor2")  editor2 에디터 디자인 영역에 작성된 내용 변경 사항을 확인합니다.

에디터에 내용을 입력 후 변경 사항을 확인할 때 유용하게 사용할 수 있습니다.
