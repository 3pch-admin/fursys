K Editor :: setInsertHTMLEx

void setInsertHTMLEx(html, position, editor_id)

에디터 디자인 영역에서 커서가 있는 위치 혹은 작성영역 제일 뒤에 html 소스를 삽입합니다.

return value

없음.

parameters

html  에디터 디자인 영역에 삽입할 html 소스입니다.
position  html 소스를 삽입할 위치를 의미합니다.
editor_id  html 소스를 입력할 에디터의 id를 의미합니다.

remarks

position 값은 0으로 설정 시 커서가 있는 위치에 html소스 삽입, 1로 설정 시 작성영역 제일 뒤에 삽입합니다.

에디터를 1개만 생성했을 경우 editor_id를 입력하지 않아도 html 소스를 삽입할 수 있습니다.
예) id가 editor1인 에디터를 생성했을 경우
     RAONKEDITOR.setInsertHTMLEx(html, 1, 'editor1'), RAONKEDITOR.setInsertHTMLEx(html, 1)
     모두 editor1 에디터 디자인 영역에서 작성영역 제일 뒤에 html 소스를 삽입합니다.

에디터를 2개 이상 생성했을 경우 editor_id를 반드시 입력해야 합니다.
예) id가 editor1, editor2인 에디터를 생성했을 경우
     RAONKEDITOR.setInsertHTMLEx(html, 0, 'editor1')  editor1 에디터 디자인 영역에서 작성영역 제일 뒤에 html 소스를 삽입합니다.
     RAONKEDITOR.setInsertHTMLEx(html, 1, 'editor2')  editor2 에디터 디자인 영역에서 커서위치에 html 소스를 삽입합니다.
