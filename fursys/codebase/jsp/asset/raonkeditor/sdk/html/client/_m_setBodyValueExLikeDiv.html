K Editor :: setBodyValueExLikeDiv

void setBodyValueExLikeDiv(value, editor_id)

div 태그를 에디터 디자인 영역에 <body> 태그로 입력합니다.
div 태그의 배경화면을 에디터에 넣고 싶을 때 유용합니다.
작성한 글을 보여주는 부분이 div 태그일 때 유용합니다.
(이 함수를 사용하시려면 글을 저장할 때 getBodyValueExLikeDiv 함수를 사용해야 합니다.)

return value

없음.

parameters

value  에디터 디자인 영역에 설정할 div 태그 소스를 의미합니다.
editor_id  div 태그 소스를 입력할 에디터의 id를 의미합니다.

remarks

에디터를 1개만 생성했을 경우 parameter를 입력하지 않아도 div 태그 소스를 입력할 수 있습니다.
예) id가 editor1인 에디터를 생성했을 경우
     RAONKEDITOR.setBodyValueExLikeDiv(div_source, 'editor1'), RAONKEDITOR.setBodyValueExLikeDiv(div_source)
     모두 editor1 에디터 디자인 영역에 div 태그로 소스를 입력합니다.

에디터를 2개 이상 생성했을 경우 parameter를 반드시 입력해야 합니다.
예) id가 editor1, editor2인 에디터를 생성했을 경우
     RAONKEDITOR.setBodyValueExLikeDiv(div_source, 'editor1')  editor1 에디터 디자인 영역에 div 태그로 소스를 입력합니다.
     RAONKEDITOR.setBodyValueExLikeDiv(div_source, 'editor2')  editor2 에디터 디자인 영역에 div 태그로 소스를 입력합니다.
