K Editor :: Mode

XML : <mode>.....</mode>, Script : Mode

Script 설정이 우선 적용됩니다.
에디터를 편집모드, 보기모드로 설정합니다.

remarks

기본값은 "edit" 편집모드로 설정됩니다.
"edit or 빈값"이면 "편집모드"이고, "view"로 설정시 보기모드 입니다.

sample code

에디터를 보기모드로 설정합니다.

XML

.....
<setting>
    .....
    <mode>view</mode>
    .....
</setting>
.....

Script

<script type="text/javascript" src="raonkeditor/js/raonkeditor.js"></script>
	
<!-- ..... 생략 ..... -->
