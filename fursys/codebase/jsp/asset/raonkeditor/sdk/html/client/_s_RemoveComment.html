K Editor :: RemoveComment

XML : <remove_comment>.....</remove_comment>, Script : RemoveComment

Script 설정이 우선 적용됩니다.
에디터에 내용이 들어올 때 주석 삭제 여부를 설정합니다.

remarks

기본값은 "1" 주석 삭제, "0" 으로 설정 시 주석을 미삭제 합니다.

sample code

에디터에 들어온 내용 중 주석을 미삭제합니다.

XML

.....
<setting>
    .....
    <remove_comment>0</remove_comment>
    .....
</setting>
.....
