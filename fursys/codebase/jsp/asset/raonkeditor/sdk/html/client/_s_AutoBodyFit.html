K Editor :: AutoBodyFit

XML : <auto_body_fit>.....</auto_body_fit>, Script : AutoBodyFit

Script 설정이 우선 적용됩니다.
가로 눈금자 사용 설정 시, 눈금자 포인터 위치 값(픽셀) 기준으로 줄내림 사용여부를 설정합니다.

remarks

기본값은 "0" 으로 설정 시 사용 안함이고, "1" 로 설정 시 줄내림 사용으로 설정합니다.
