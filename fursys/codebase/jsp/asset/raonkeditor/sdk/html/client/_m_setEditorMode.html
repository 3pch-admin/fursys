K Editor :: SetEditorMode

void SetEditorMode(mode, editor_id)

에디터를 view 또는 edit 모드로 변경합니다.

return value

없음.

parameters

mode  설정할 모드를 의미합니다.
editor_id  에디터의 id를 의미합니다.

remarks

보기모드와 편집모드를 동시에 사용할 경우 유용합니다.

sample code

<!-- ..... 생략 ..... -->

<script type="text/javascript" src="raonkeditor/js/raonkeditor.js"></script>	
<script type="text/javascript">

    function setViewMode() {
        // 에디터를 view 모드로 설정합니다.
        RAONKEDITOR.SetEditorMode('view', 'editor1');
    }

    function setEditMode() {
        // 에디터를 edit 모드로 설정합니다.
        RAONKEDITOR.SetEditorMode('edit', 'editor1');
    }

</script>	

<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">
  
    <!-- 에디터 생성 -->  
    <script type="text/javascript">
         new RAONKEditor({Id:'editor1'});
    </script>

</div>
