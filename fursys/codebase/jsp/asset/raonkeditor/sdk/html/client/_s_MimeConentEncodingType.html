K Editor :: MimeConentEncodingType

Script : MimeConentEncodingType

MIME 인코딩 시에 인코딩 방법을 설정합니다.

remarks

인코딩 방법 설정값은
"1"로 설정 시 "7bit", "2"로 설정 시 "8bit", "3"으로 설정 시 "base64", "4"로 설정 시 "quoted-printable" 입니다.
기본값은 "3"으로 "base64" 입니다.

sample code

<script type="text/javascript" src="raonkeditor/js/raonkeditor.js"></script>	
 
<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">  
    <script type="text/javascript">

        var raonkParam = {
            .....
            // 인코딩 방법을 quoted-printable 로 설정합니다.
            MimeConentEncodingType: '4'
        }

        new RAONKEditor(raonkParam);
    </script>
</div>
