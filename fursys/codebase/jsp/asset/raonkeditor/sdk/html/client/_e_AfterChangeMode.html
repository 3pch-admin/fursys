K Editor :: RAONKEDITOR_AfterChangeMode

void RAONKEDITOR_AfterChangeMode(editorId, paramObj)

에디터 보기화면(statusBar) 변경 후 이벤트가 발생합니다.

return value

없음.

parameters

paramObj.strNewMode  변경 후 보기화면입니다.
paramObj.strBeforeMode  변경 전 보기화면입니다.
editorId  에디터의 id입니다.

remarks

보기화면(statusBar) 모드명은 다음과 같습니다.
디자인 - "design"
HTML - "source"
미리보기 - "preview"
TEXT - "text"
