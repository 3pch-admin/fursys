K Editor :: FontSizeList

XML : <font_size>.....</font_size>, Script : FontSizeList

Script 설정이 우선 적용됩니다.
에디터 폰트 크기 메뉴에 노출할 크기 리스트를 설정합니다.

remarks

XML:font_size  필요 시 <size>.....</size> 항목을 추가하여 설정합니다.
Script:FontSizeList  필요 시 FontSizeList: "FontSize값"에 콤마(,)로 구분하여 설정합니다.

sample code

에디터 크기 메뉴에 노출할 크기를 10pt, 11pt 으로 설정합니다.

XML

.....
<font_size>
    <size>10pt</size>
    <size>11pt</size>
</font_size>
.....

Script

<script type="text/javascript" src="raonkeditor/js/raonkeditor.js"></script>	
 
<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">  
    <script type="text/javascript">

        var raonkParam = {
            .....
            FontSizeList: '10pt,11pt'
        }

        new RAONKUpload(raonkParam);
    </script>
</div>
