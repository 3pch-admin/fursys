K Editor :: XhtmlValue

XML : <xhtml_value>.....</xhtml_value>, Script : XhtmlValue

Script 설정이 우선 적용됩니다.
소스보기나 에디터 영역의 html 소스를 가져올 때 xhtml 형식으로 가져옵니다.

remarks

"0" 으로 설정 시 사용 안함, "1" 로 설정 시 사용으로 설정합니다.
기본값은 "0" 입니다.
