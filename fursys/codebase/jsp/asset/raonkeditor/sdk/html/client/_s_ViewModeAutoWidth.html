K Editor :: ViewModeAutoWidth

XML : <view_mode_auto_width>.....</view_mode_auto_width>, Script : ViewModeAutoWidth

Script 설정이 우선 적용됩니다.
View 모드 시 에디터 안의 내용만큼 자동으로 width가 늘어납니다.

remarks

기본값은 "0" 이고 자동 width 미사용, "1"로 설정 시 사용으로 설정합니다.

sample code

에디터 자동 width를 사용으로 설정합니다.

XML

.....
<setting>
    .....
    <view_mode_auto_width>1</view_mode_auto_width>
    .....
</setting>
.....

Script

<script type="text/javascript" src="raonkeditor/js/raonkeditor.js"></script>	
 
<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">   
    <script type="text/javascript">
        var raonkParam = {
            .....
            ViewModeAutoWidth: '1'
           
        }

        new RAONKEditor(raonkParam);
    </script>
</div>
