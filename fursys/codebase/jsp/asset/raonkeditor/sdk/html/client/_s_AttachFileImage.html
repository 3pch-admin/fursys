K Editor :: AttachFileImage

XML : <attach_file_image>.....</attach_file_image>

파일 삽입 시 파일 관련 이미지 삽입을 설정합니다.

remarks

기본값은 "1" 이며 이미지 삽입 허용, "0" 으로 설정 시 이미지 삽입 허용 안함으로 설정합니다.

sample code

파일 삽입 시 파일 관련 이미지 삽입 허용 안함으로 설정합니다. 

.....
<uploader_setting>
    .....
    <attach_file_image>0</attach_file_image>
    .....
</uploader_setting>
.....
