K Editor :: UserCssUrl

XML : <user_css_url>.....</user_css_url>, Script : UserCssUrl
XML : <user_css_url always_set="....."></user_css_url>, Script : UserCssAlwaysSet

Script 설정이 우선 적용됩니다.
사용자가 만든 css를 에디터 영역에 설정합니다.
그리고 그 css를 항상 포함할지 여부를 설정합니다.

remarks

XML:user_css_url,Config:UserCssUrl  http://를 포함하는 전체 url을 입력해야 합니다.
XML:always_set,Config:UserCssAlwaysSet  기본값은 "0" 사용자가 만든 css를 에디터 영역에 항상 포함되지 않으며, "1" 설정시 항상 포함됩니다.

sample code

"http://www.raonk.com/user_css.css" 링크의 css를 에디터 영역에 설정합니다.
그리고 그 css를 항상 포함 설정합니다.

XML

.....
<setting>
    .....
    <user_css_url always_set="1">http://www.raonk.com/user_css.css</user_css_url>
    .....
</setting>
.....

Script

<script type="text/javascript" src="raonkeditor/js/raonkeditor.js"></script>	
 
<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">
