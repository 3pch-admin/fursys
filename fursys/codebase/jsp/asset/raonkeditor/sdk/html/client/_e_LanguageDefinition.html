K Editor :: RAONKEDITOR_OnLanguageDefinition

void RAONKEDITOR_OnLanguageDefinition(editorId, editorLang)

에디터 생성 시 이벤트가 발생합니다.

return value

없음.

parameters

editorId  에디터의 id입니다.
editorLang  다국어 설정값(lang)에 따른 텍스트 객체입니다.

remarks

에디터에서 노출되는 아이콘명, 메세지 등을 변경합니다.
한글, 영어, 일본어, 중국어(간체,번체)에 대한 별도의 다국어 설정이 존재합니다.
