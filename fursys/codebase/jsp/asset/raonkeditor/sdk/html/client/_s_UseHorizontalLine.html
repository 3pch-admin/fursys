K Editor :: UseHorizontalLine

XML : <use_horizontal_line>.....</use_horizontal_line>
XML : <use_horizontal_line url="....."></use_horizontal_line>, Script : UseHorizontalLine

Script 설정이 우선 적용됩니다.
XML:use_horizontal_line  세로 기준선 표시 사용여부를 설정합니다.
XML:url,Script:UseHorizontalLine  기준선 이미지 경로를 설정합니다.

remarks

XML:use_horizontal_line  "0" 으로 설정 시 사용 안함, "1"로 설정 시 세로 기준선 표시 사용으로 설정합니다.
XML:url,Config:UseHorizontalLine  http://를 포함한 전체 이미지 경로를 콤마(,)로 구분하여 입력합니다.

sample code

세로 기준선 표시 사용여부를 사용으로 설정합니다.
기준선 이미지 경로를 http://www.raonk.com/custom_horizontal_line.png 으로 설정합니다.

XML

.....
<setting>
    .....
    <use_horizontal_line url="http://www.raonk.com/custom_horizontal_line.png">1</use_horizontal_line>
    .....
</setting>
.....
