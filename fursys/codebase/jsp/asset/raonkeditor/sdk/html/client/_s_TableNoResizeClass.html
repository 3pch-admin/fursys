K Editor :: TableNoResizeClass

XML : <no_resize_class>.....</no_resize_class>
Script : TableNoResizeClass

Script 설정이 우선 적용됩니다.
테이블에 resize 사용여부를 설정합니다.

remarks

설정된 class명을 테이블에 주면 테이블 resize를 미사용으로 설정합니다.

sample code

"noResize" 클래스명이 포함된 테이블은 resize를 미사용으로 설정합니다.

XML

.....
<setting>
    .....
    <table_property>
        .....
        <no_resize_class>noResize</no_resize_class>
        .....
    </table_property>
    .....
</setting>
.....

Script

<script type="text/javascript" src="raonkeditor/js/raonkeditor.js"></script>	
 
<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">  
    <script type="text/javascript">
        var raonkParam = {
            .....
            TableNoResizeClass: 'noResize'
        }

        new RAONKEditor(raonkParam);
    </script>
</div>
