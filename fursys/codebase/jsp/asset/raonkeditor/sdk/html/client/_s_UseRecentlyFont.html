K Editor :: UseRecentlyFont

XML : <font_family use_recently_font="....."></font_family>, Script : UseRecentlyFont

Script 설정이 우선 적용됩니다.
에디터 글꼴 메뉴에 최근사용 폰트 추가 여부를 설정합니다.
제일 상단에 최대 5개까지 표현됩니다.

remarks

기본값은 "0" 이고 최근사용 폰트 추가 미설정, "1" 로 설정 시 최근사용 폰트 추가를 설정 합니다.

sample code

에디터 글꼴 메뉴에 최근사용 폰트 추가 설정 합니다.

XML

.....
<font_family use_recently_font="1">
    .....
</font_family>
.....

Script

<script type="text/javascript" src="raonkeditor/js/raonkeditor.js"></script>	
 
<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">    
    <script type="text/javascript">
        var raonkParam = {
            .....
            UseRecentlyFont: '1'
        }

        new RAONKEditor(raonkParam);
    </script>
</div>
