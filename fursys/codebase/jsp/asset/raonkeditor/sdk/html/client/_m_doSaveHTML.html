K Editor :: doSaveHTML

void doSaveHTML(strContent, strDefaultFileName, strCharSet, editor_id)

에디터에 작성한 소스를 html파일로 저장합니다.

return value

없음.

parameters

strContent  저장할 html 소스입니다.
strDefaultFileName  저장할 기본 html 파일명입니다.
strCharSet  저장할 문자 형태입니다.
editor_id  접근할 에디터의 id 입니다.

remarks

strCharSet에 "utf-8"을 입력하면 UTF-8 형태로, 빈값("")이면 ANSI 형태로 저장합니다.

sample code

<script type="text/javascript" src="keditor/js/raonkeditor.js"></script>	
<script type="text/javascript">

    function saveHTML(){
        var strContent = RAONKEDITOR.getBodyValue('editor1');
        var strDefaultFileName = "raonk";
        var strCharSet = "utf-8";

        RAONKEDITOR.doSaveHTML(strContent, strDefaultFileName, strCharSet, "editor1");
    }

</script>
