Script : Doctype

에디터 Doctype를 설정합니다.

remarks

DocType 설정 시 우선 적용 됩니다.

DocType의 종류는
None, XHTML 1.1, XHTML 1.0 Transitional, XHTML 1.0 Strict, XHTML 1.0 Frameset,
HTML 5, HTML 4.01 Transitional, HTML 4.01 Strict, HTML 4.01 Frameset, HTML 3.2, HTML 2.0 입니다.
