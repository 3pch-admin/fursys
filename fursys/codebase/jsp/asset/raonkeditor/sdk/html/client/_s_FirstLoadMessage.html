K Editor :: FirstLoadMessage
K Editor :: FirstLoadType

Script : FirstLoadType, Script : FirstLoadMessage

에디터 생성 후 디자인영역에 텍스트, html 등을 띄울 수 있습니다.

remarks

FirstLoadMessage을 설정하기 위해서는 FirstLoadType을 설정해야 합니다.

FirstLoadType에는
<body>태그 내부 소스 "bodyvalue", <body>태그 포함 소스 "bodyvalueex",
<div>태그 포함 소스(디자인 영역에 <body>태그로 입력) "bodyvalueexlikediv",
<html>태그 내부 소스 "htmlvalue", <html>태그 포함 소스 "htmlvalueex",
url "url", 비우기 "empty" 가 있으며 상황에 맞는 값을 설정하시면 됩니다.

FirstLoadType을 설정 하신 후 FirstLoadMessage에 html소스 또는 url을 입력하시면 됩니다.

DefaultMessage와 loadHtmlValueExFromURL메소드와 함께 유용하게 사용할 수 있습니다.

sample code

<script type="text/javascript" src="raonkeditor/js/raonkeditor.js"></script>	
 
<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">    
    <script type="text/javascript">

    // 에디터 생성 시 디자인 영역에 DefaultMessage가 나타나고,
    // 디자인 영역을 클릭하거나 키보드를 누르면 FirstLoadMessage의 html페이지가 디자인영역에 나타납니다.
           var raonkParam = {
            .....
            DefaultMessage: '<span>휴가 계획서를 작성해 주세요.</span>',
            FirstLoadType: 'url',
            FirstLoadMessage: 'http://www.raonk.com/raonkeditor/pages/forms/vacation.html'
        }
        new RAONKEditor(raonkParam);
      
    </script>
</div>
