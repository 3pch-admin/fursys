K Editor :: InitXml

Script : InitXml

에디터 생성시 로드할 XML파일을 설정합니다.

remarks

기본값은 "raonkeditor.config.xml" 입니다.
필요 시 InitXml: 'xml 파일명' 으로 설정합니다.
파일 경로는 "raonkeditor/config" 입니다.

sample code

<script type="text/javascript" src="raonkeditor/js/raonkeditor.js"></script>	

<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        // 에디터를 example.xml을 로드하여 생성합니다.
        var raonkParam = {
            .....
            InitXml: 'example.xml'
        }

        new RAONKEditor(raonkParam);
    </script>
</div>
