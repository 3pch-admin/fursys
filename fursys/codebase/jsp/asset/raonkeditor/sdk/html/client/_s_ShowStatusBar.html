K Editor :: ShowStatusBar

XML : <show_statusbar>.....</show_statusbar>, Script : StatusBar

Script 설정이 우선 적용됩니다.
에디터의 보기화면(statusBar)은 "디자인", "HTML", "미리보기", "TEXT" 탭으로 구성되어 있습니다.
이 설정은 보기화면(statusBar)을 보이거나 숨기는 기능을 설정합니다.
사용 용도에 따라서 상태바 숨김으로 설정을 원할 때 사용하면 유용 할 수 있습니다.

remarks

기본값은 "1" 이고, "0" 으로 설정시 상태바를 표시하지 않습니다.

sample code

에디터 보기화면(statusBar)을 숨김으로 설정합니다.


XML

.....
<setting>
    .....
    <show_statusbar>0</show_statusbar>
    .....
</setting>
.....

Script

<script type="text/javascript" src="raonkeditor/js/raonkeditor.js"></script>
	
<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        var raonkParam = {
            .....
            ShowStatusBar: '0',
        }

        new RAONKEditor(raonkParam);
    </script>
</div>
