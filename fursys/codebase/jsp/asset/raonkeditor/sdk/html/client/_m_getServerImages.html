K Editor :: GetServerImages

string GetServerImages(imgURL, editorName)

에디터 디자인 영역에 있는 서버 이미지 태그 경로를 가져옵니다.
문서에서 사용중인 모든 서버 이미지명을 알기 위하여 사용합니다.
문서등록 시 파일명 Guid 형태 변환옵션을 사용하는 것이 좋습니다.

return value

아래와 같은 형식으로 이미지 개수만큼 나열되어 스트링으로 가져옵니다.
이미지전체경로 + (구분자 : 0x0b) + 이미지명 + (구분자 : 0x0c) + ... +

parameters

imgURL  가져올 이미지의 서버 url입니다.
             예) 'http://' - 'http://' 로 시작하는 경로의 이미지만 찾습니다.
             예) 'http://www.raonk.com/' - 'http://www.raonk.com/' 로 시작하는 경로의 이미지만 찾습니다.
editorName  이미지를 가져올 에디터의 id를 의미합니다.

remarks

에디터를 1개만 생성했을 경우 editorName을 입력하지 않아도 이미지 태그를 가져올수 있습니다.
예) id가 editor1인 에디터를 생성했을 경우
     RAONKEDITOR.GetServerImages(imgURL, "editor1"), RAONKEDITOR.GetServerImages(imgURL)  
     모두 editor1 에디터 디자인 영역의 이미지 태그 경로를가져옵니다.

에디터를 2개 이상 생성했을 경우 editorName을 반드시 입력해야 합니다.
예) id가 editor1, editor2인 에디터를 생성했을 경우
     RAONKEDITOR.GetServerImages(imgURL, "editor1")  editor1 에디터 디자인 영역의 이미지 태그 경로를 가져옵니다.
     RAONKEDITOR.GetServerImages(imgURL, "editor2")  editor2 에디터 디자인 영역의 이미지 태그 경로를 가져옵니다.

<script type="text/javascript">

    function getServerImages() {
        // id가 editor1인 에디터 디자인 영역에서 경로가 http://로 시작하는 이미지 태그를 가지고 온 후 "\u000c"로 구분합니다.
        var sBodyImage = RAONKEDITOR.GetServerImages("http://", "editor1").split("\u000c");
        var sBodyURL = "";
        for(var i = 0 ; i < sBodyImage.length; i++) {
            // 이미지당 2개씩 넘어오는 변수를 자릅니다
            sBodyURL = sBodyImage[i].split("\u000b");
            // 이미지 전체경로와 이미지명을 얻을수 있습니다
            alert(sBodyURL[0] + " / " + sBodyURL[1]);
        }
    }

</script>
