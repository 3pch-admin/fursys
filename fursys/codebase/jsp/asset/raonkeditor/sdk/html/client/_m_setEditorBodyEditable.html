K Editor :: SetEditorBodyEditable

void SetEditorBodyEditable(bEditable, editor_id)

에디터 body 영역 안에 작성 가능 여부를 설정 합니다.

return value

없음.

parameters

bEditable  에디터 body 영역안에 작성 가능 여부를 의미합니다.
editor_id  에디터의 id를 의미합니다.

remarks

작성이 가능 하면 true이고, 작성이 가능 하지 않으면 false 입니다.

sample code

<!-- ..... 생략 ..... -->

<script type="text/javascript" src="raonkeditor/js/raonkeditor.js"></script>	
<script type="text/javascript">

    function setEditable() {
        // id가 editor1인 에디터 body 영역 안에 작성 가능으로 설정 합니다.
        RAONKEDITOR.SetEditorBodyEditable(true, editor1);
    }

</script>
