K Editor :: UserHelpUrl

Script : UserHelpUrl

에디터 사용자 매뉴얼(help 폴더) 경로를 설정합니다.

remarks

http:// 포함하는 전체 url 를 입력합니다.

sample code

<script type="text/javascript" src="roankeditor/js/raonkeditor.js"></script>	
 
<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">    
    <script type="text/javascript">

        var paramObj = {
            Id: 'editor1',          
            // help 폴더 경로를 설정합니다.
            UserHelpUrl: 'http://www.raonk.com/help/'
        };
        
        new RAONKEditor(paramObj);

    </script>       
</div>
