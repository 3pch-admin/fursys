K Editor :: DragResize

XML : <drag_resize>.....</drag_resize>, Script : config.DragResize
XML : <drag_resize apply_browser="....."></drag_resize>, Script : config.DragResizeApplyBrowser
XML : <drag_resize apply_div_class="....."></drag_resize>, Script : config.DragResizeApplyDivClass

XML:drag_resize,Script:DragResize  에디터 또는 브라우저의 기본 리사이즈 사용여부를 설정합니다.
XML:apply_browser,Script:DragResizeApplyBrowser  리사이즈 기능을 사용할 브라우저를 설정합니다.
XML:apply_div_class,Script:DragResizeApplyDivClass  사용자가 지정한 class명의 div태그를 에디터에서 제공하는 리사이즈기능으로 컨트롤할 수 있습니다.

remarks

XML:drag_resize,Script:DragResize  기본값은 "0" 이고 브라우저 기본 리사이즈를 사용, "1"로 설정 시 에디터에서 제공하는 리사이즈를 사용합니다.
"1" 일 경우 XML:apply_browser,Config:DragResizeApplyBrowser,
XML:apply_div_class,Script:DragResizeApplyDivClass 설정이 가능합니다.
XML:apply_browser,Script:DragResizeApplyBrowser  ie,edge,firefox,opera,chrome,safari 설정값 중 설정된 브라우저만 리사이즈를 사용합니다.
빈값일 경우 모든 브라우저가 설정됩니다.
XML:apply_div_class,Script:DragResizeApplyDivClass  기본값은 모든 div 리사이즈기능을 사용합니다.

sample code

에디터에서 제공하는 리사이즈를 사용 설정합니다.
에디터에서 제공하는 리사이즈를 ie,firefox,opera 브라우저만 사용합니다.
"resizeClass" class명의 div태그를 에디터에서 제공하는 리사이즈기능으로 컨트롤 가능하도록 설정합니다.

XML

.....
<setting>
    .....
    <drag_resize apply_browser="ie,firefox,opera" apply_div_class="resizeClass">1</enter_tag>
    .....
</setting>
.....
