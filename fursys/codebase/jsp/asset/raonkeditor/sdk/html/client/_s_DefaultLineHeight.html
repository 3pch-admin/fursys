K Editor :: DefaultLineHeight

XML : <line_height>.....</line_height>, Script : DefaultLineHeight
XML : <line_height mode="....."></line_height>, Script : LineHeightMode

에디터 로드 시 사용할 줄간격과 줄간격 Mode(% or 배수)를 설정합니다.

remarks

XML:line_height,Script:DefaultLineHeight  기본 줄간격은 "1.2" 입니다.
XML:mode,Script:LineHeightMode  기본 Mode는 "2" 배율, "1"로 설정시 %로 설정됩니다.

sample code

<script type="text/javascript" src="keditor/js/raonkeditor.js"></script>	
 
<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">    
    <script type="text/javascript">

        var paramObj = {
            Id: 'editor1',          
            // 에디터 기본 줄간격을 1.5 으로 설정합니다.
            DefaultLineHeight: '1.5'
        };
        
        new RAONKEditor(paramObj);

    </script>       
</div>
