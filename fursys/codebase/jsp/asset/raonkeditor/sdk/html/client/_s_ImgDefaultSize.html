K Editor :: ImgDefaultSize

XML : <img_default_size width="">.....</img_default_size>
XML : <img_default_size height="">.....</img_default_size>

이미지 다이얼로그창의 너비, 높이 기본값을 설정합니다.

remarks

예) width="100" / height="100" 설정 시
너비는 "100" / 높이는 "100" 으로 표시 합니다.

sample code

이미지 다이얼로그창의 너비를 200으로 설정합니다.
이미지 다이얼로그창의 높이를 200으로 설정합니다.

XML

.....
<setting>
    .....
    <img_default_size width="200" height="200"></img_default_size>
    .....
</setting>
.....
