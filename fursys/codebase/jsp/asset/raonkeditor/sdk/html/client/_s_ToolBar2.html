K Editor :: ToolBar2

XML : <tool_bar_2>.....</tool_bar_2>

에디터의 하단(서식) 툴바의 메뉴를 표시하거나 숨기는 기능을 설정합니다.

remarks

하단(서식) 툴바의 메뉴
메뉴명 command 명
서식 콤보 formatting
폰트 콤보 font_family
폰트 크기 콤보 font_size
줄간격 line_height
진하게 bold
이탤릭 italic
밑줄 underline
취소선 strike_through
위 첨자 superscript
아래 첨자 subscript
서식 지우기 remove_format
css 지우기 remove_css
글자색 font_color
글자 배경색 font_bg_color
번호 매기기 list_number
글머리 기호 list_bullets
왼쪽 정렬 align_left
가운데 정렬 align_center
오늘쪽 정렬 align_right
양쪽 정렬 align_full
들여 쓰기 indent
내어 쓰기 outdent
<remove_item />에 하위 앨리먼트로 메뉴 command 명을 지정해 주면 해당 메뉴를 표시하지 않습니다.
예)<remove_item><item>indent</item><item>outdent</item></remove_item>

sample code

하단(서식) 툴바의 메뉴에 서식 콤보,폰트 콤보,폰트 크기 콤보,줄간격,구분선,진하게,이탤릭,밑줄..... 메뉴를 표시 설정 합니다.
