K Editor :: EditorBorder

Script : EditorBorder

에디터 전체영역 보더 라인을 보이거나 숨기는 설정합니다.

remarks

기본값은 "1" 이고, "0" 로 설정 시 보더 라인 숨김으로 설정합니다.

sample code

<script type="text/javascript" src="raonkeditor/js/raonkeditor.js"></script>	
 
<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">    
    <script type="text/javascript">

        // 에디터 전체 영역 보더 라인을 숨김으로 설정합니다.
        var raonkParam = {
            .....
            EditorBorder: '0'
        }
        
        new RAONKEditor(raonkParam);

    </script>       
</div>
