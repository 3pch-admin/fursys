K Editor :: ClearUserCssUrl

void ClearUserCssUrl(bool, editorName)

에디터에 적용된 css 경로를 제거합니다.

return value

없음.

parameters

bool  모두 제거할 것인지, 처음 로드한 css는 제외하고 제거할 것인지 설정합니다.
editorName  접근할 에디터의 id 입니다.

remarks

bool 값에 true를 입력하면 에디터에 적용된 css 경로를 모두 삭제합니다.
bool 값에 false를 입력하면 환경설정 xml에서 user_css_url 부분에 설정된 css는 제외하고 모두 삭제합니다.
