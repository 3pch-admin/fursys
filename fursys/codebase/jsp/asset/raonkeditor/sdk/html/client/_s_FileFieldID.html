K Editor :: FileFieldID

Script : FileFieldID

파일 업로드에 사용되는 File 태그의 name을 설정합니다.

remarks

기본값은 'Filedata' 로 설정됩니다.
필요시 Script: FileFieldID = "newName" 으로 설정합니다.

sample code


<script type="text/javascript" src="raonkeditor/js/raonkeditor.js"></script>	
 
<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">  
        <script type="text/javascript">

            // 파일 업로드 File 태그의 name을 raonk로 변경합니다.
        var raonkParam = {
            .....
            FileFieldID: 'raonk'
        }
            new RAONKEditor(raonkParam);

        </script>       
</div>
