K Editor :: ScrollOverflow

XML : <scroll_overflow>.....</scroll_overflow>, Script : ScrollOverflow

Script 설정이 우선 적용됩니다.
에디터 영역에 스크롤바를 설정합니다.

remarks

기본값은 "0" 이고 스크롤 "auto" 입니다.
"1" 로 설정 시 스크롤 "X축만 설정" 합니다.
"2" 로 설정 시 스크롤 "Y축만 설정" 합니다
"3" 로 설정 시 스크롤 "X,Y축 모두 설정" 합니다
"4" 로 설정 시 스크롤 "X축 미설정" 합니다.
"5" 로 설정 시 스크롤 "Y축 미설정" 합니다.
"6" 로 설정 시 스크롤 "X,Y축 모두 미설정" 합니다.

get,setBodyValue API를 사용할 때만 가능하며, 그 이외 API 사용 시 body 태그에 style:overflow가 유지됩니다.

sample code

에디터 영역에 스크롤바를 x축만 설정합니다.

XML

.....
<setting>
    .....
    <scroll_overflow>1</scroll_overflow>
    .....
</setting>
.....

Script

<script type="text/javascript" src="raonkeditor/js/raonkeditor.js"></script>	
 
<!-- ..... 생략 ..... -->

    <script type="text/javascript">

        }

        new RAONKEditor(raonkParam);
    </script>
</div>
