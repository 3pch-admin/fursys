K Editor :: IgnoreSameEditorName

Script : IgnoreSameEditorName

에디터 명 중복시 임의로 변경 생성하도록 설정합니다.

remarks

기본값은 "0" 이고, "1" 로 설정 시 중복된 에디터 명이 발생 할 경우 에디터에서 임의로 변경 합니다. (단, 동적 생성 시)

sample code

<script type="text/javascript" src="raonkeditor/js/raonkeditor.js"></script>	

<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        // 중복 된 에디터 명이 발생 할 경우 임의로 변경 설정합니다.
        var raonkParam = {
            .....
            IgnoreSameEditorName: '1'
        }

        new RAONKEditor(raonkParam);
    </script>
</div>
