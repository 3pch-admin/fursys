K Editor :: Lang

XML : <lang>.....</lang>, Script : Lang

Script 설정이 우선 적용됩니다.
에디터에 로드 할 언어를 설정합니다.

remarks

기본값은 "ko-kr : 한국어" 입니다.

언어코드는
ko-kr : 한국어(대한민국)
en-us : 영어(미국)
zh-cn : 중국어(중국)
zh-tw : 중국어(대만)
ja-jp : 일본어

sample code

에디터 언어를 en-us로 설정합니다.

XML

.....
<setting>
    .....
    <lang>en-us</lang>
    .....
</setting>
.....

Script

<script type="text/javascript" src="raonkeditor/js/raonkeditor.js"></script>
	
<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        var raonkParam = {
            .....

        }

    </script>
</div>
