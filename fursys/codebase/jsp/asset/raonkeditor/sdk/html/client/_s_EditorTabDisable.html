K Editor :: EditorTabDisable

XML : <editor_tab_disable>.....</editor_tab_disable>, Script : EditorTabDisable

Script 설정이 우선 적용됩니다.
디자인영역에서 tab키 입력 시 공백 입력을 설정합니다.

remarks

기본값은 "0" 이고 tab키 입력 시 에디터 디자인영역에 공백 입력, "1" 로 설정 시 다음 컨트롤로 포커스를 이동합니다.

sample code

에디터에서 tab키를 입력하면 다음 컨트롤로 포커스 이동을 설정합니다.

XML

.....
<setting>
    .....
    <editor_tab_disable>1</editor_tab_disable>
    .....
</setting>
.....

<script type="text/javascript" src="raonkeditor/js/raonkeditor.js"></script>
