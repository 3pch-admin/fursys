K Editor :: getDeletedElementsUrl

string getDeletedElementsUrl(editorName)

에디터에서 삭제한 elements의 url를 가져옵니다.

return value

삭제된 elements의 url을 리턴 합니다.

parameters

editorName  삭제한 elements의 url를 가져올 에디터의 id를 의미합니다.

remarks

없음.

sample code

<!-- ..... 생략 ..... -->

<script type="text/javascript" src="raonkeditor/js/raonkeditor.js"></script>	
<script type="text/javascript">

    function getDeletedElementsUrl(){
        // id가 editor1인 에디터에서 삭제한 elements의 url를 가져옵니다.
        var deletedElementsUrl = RAONKEDITOR.getDeletedElementsUrl('editor1');

        if (deletedElementsUrl.length > 0) {
            var result = '';

            for (var i = 0; i < deletedElementsUrl.length; i++) {
                result += deletedElementsUrl[i] + '\n';
            }
            alert(result);
        }
    }

</script>
