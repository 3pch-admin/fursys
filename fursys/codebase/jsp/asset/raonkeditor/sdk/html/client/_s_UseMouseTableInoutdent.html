K Editor :: UseMouseTableInoutdent

XML : <use_mouse_inoutdent>.....</use_mouse_inoutdent>
Script : UseMouseTableInoutdent

Script 설정이 우선 적용됩니다.
마우스로 테이블 들여쓰기,내어쓰기 사용여부를 설정합니다.

remarks

기본값은 "0" 이고 마우스로 테이블 들여쓰기,내어쓰기 사용하지 않음으로 설정됩니다.
"1"로 설정 시 마우스로 테이블 들여쓰기,내어쓰기 사용함으로 설정됩니다.

sample code

마우스로 테이블 들여쓰기,내어쓰기 사용여부를 사용함으로 설정합니다.

XML

.....
<setting>
    .....
    <table_property>
        .....
        <use_mouse_inoutdent>1</use_mouse_inoutdent>
        .....
    </table_property>
    .....
</setting>
.....

<script type="text/javascript" src="raonkeditor/js/raonkeditor.js"></script>
