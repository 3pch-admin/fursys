K Editor :: RAONKEDITOR_CreationComplete

void RAONKEDITOR_CreationComplete(editor_id)

에디터가 로드 완료(작업준비) 되었을 때 발생합니다.

return value

없음.

parameters

editor_id   생성완료된 에디터 ID 입니다.

remarks

에디터가 작업 준비가 완료되었을 때 발생하므로 여러가지 초기값을 설정하려면 이곳에서 설정을 합니다.
