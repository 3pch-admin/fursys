K Editor :: setEditorChangeMode

void setEditorChangeMode(mode, editor_id)

에디터의 상태 바는 "디자인", "HTML", "미리 보기", "TEXT" 탭으로 구성되어 있습니다.
이 설정은 상태바의 탭을 이동하는 기능을 설정합니다.

return value

없음.

parameters

mode  에디터에서 이동할 상태바 탭의 값을 의미합니다.
editor_id  접근할 에디터의 id를 의미합니다.

remarks

사용 용도에 따라서 에디터의 상태 바의 탭을 이동하기를 원할 때 사용하면 유용 할 수 있습니다.
"design" 이면 디자인, "source" 이면 HTML, "preview" 이면 미리 보기, "text"이면 TEXT 탭으로 이동합니다.

sample code

<!-- ..... 생략 ..... -->

<script type="text/javascript" src="raonkeditor/js/raonkeditor.js"></script>

    function setEditorChangeMode(){
        // 에디터 상태바를 HTML 탭으로 이동합니다.
        RAONKEDITOR.setEditorChangeMode('source', 'editor1');
    }

</script>
