K Editor :: EnterTag

XML : <enter_tag>.....</enter_tag>, Script : EnterTag

Script 설정이 우선 적용됩니다.
Enter 키를 눌렀을 때 사용하게 될 태그를 설정합니다.

remarks

기본값은 "p" 태그이며, "br" 태그 등을 지정합니다.

sample code

에디터에서 Enter 키를 눌렀을 때 사용할 태그를 br 태그로 설정합니다.

XML

.....
<setting>
    .....
    <enter_tag>br</enter_tag>
    .....
</setting>
.....

<script type="text/javascript" src="raonkeditor/js/raonkeditor.js"></script>
