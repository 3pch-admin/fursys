K Editor :: DefaultFontSize

XML : <font_size>.....</font_size>, Script : DefaultFontSize

Script 설정이 우선 적용됩니다.
에디터 로드 시 사용할 기본 폰트 크기를 설정합니다.

remarks

기본 폰트는 "12pt" 입니다.
필요시 DefaultFontSize: '폰트크기' 로 설정합니다.

sample code

XML

.....
<setting>
    .....
    <font_size>10pt</font_size>
    .....
</setting>
.....
Script

<script type="text/javascript" src="keditor/js/raonkeditor.js"></script>	
 
<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">    
    <script type="text/javascript">

        var paramObj = {
            Id: 'editor1',          
            // 에디터 기본 폰트를 10pt 으로 설정합니다.
            DefaultFontSize: '10pt'
        };
        
        new RAONKEditor(paramObj);

    </script>       
</div>
