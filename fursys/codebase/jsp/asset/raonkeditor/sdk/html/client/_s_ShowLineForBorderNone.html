K Editor :: ShowLineForBorderNone

XML : <show_line_for_border_none>.....</show_line_for_border_none>
Script : ShowLineForBorderNone

Script 설정이 우선 적용됩니다.
테이블에 선이 없을 경우(선두께:0) 점선 사용여부를 설정합니다.

remarks

기본값은 "0" 이고 선이 보이지 않음으로 설정됩니다.
"1"로 설정 시 점선으로 설정됩니다.
"2"로 설정 시 점선으로 설정되고, 붙여 넣기 할 때도 점선으로 설정됩니다.

sample code

테이블에 선이 없을 경우 점선으로 설정되고, 붙여 넣기 할 때도 점선으로 설정합니다.

XML

.....
<setting>
    .....
    <table_property>
        .....
        <show_line_for_border_none>2</show_line_for_border_none>
        .....
    </table_property>
    .....
</setting>
.....

<script type="text/javascript" src="raonkeditor/js/raonkeditor.js"></script>	
 
<!-- ..... 생략 ..... -->
