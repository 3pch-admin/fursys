XML : <context_menu_disable>.....</context_menu_disable>

에디터 안에서 마우스 우클릭 시 메뉴 표시 사용여부를 설정합니다.

remarks

"0" 으로 설정 시 표시, "1"로 설정 시 우클릭 메뉴 사용 안함으로 설정합니다.
기본값은 "0" 입니다.

sample code
