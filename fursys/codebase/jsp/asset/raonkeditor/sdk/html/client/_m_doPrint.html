[에이전트 전용]

K Editor :: doPrint

void doPrint(strHeader, strFooter, editor_id)

에디터에 작성한 소스를 프린트 합니다.

return value

없음.

parameters

strHeader  프린트 머리글 부분에 나타낼 string 입니다.
strFooter  프린트 바닥글 부분에 나타낼 string 입니다.
editor_id  접근할 에디터의 id 입니다.

remarks

아래 내용은 strHeader, strFooter에 넣을 수 있는 값 입니다.
     &w : 페이지 제목
     &u : page URL
     &d : 약식 날짜
     &D : 긴 형식 날짜
     &t : 시간
     &T : 24시간 형식 시간
     &p : 페이지 번호
     &P : 전체 페이지 수
     &b + value: value를 우측정렬
     &b + value + &b: value를 가운데정렬

        // 페이지 아래에는 "바닥글"이라는 string이 가운데에 표시되어 프린트 됩니다.
