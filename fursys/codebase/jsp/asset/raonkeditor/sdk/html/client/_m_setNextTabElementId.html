K Editor :: SetNextTabElementId

void SetNextTabElementId(strId, editorName)

tab키로 포커스를 이동하여 에디터 바깥으로 포커스가 이동할 때 이 함수로 지정한 id값을 가진 객체로 포커스가 이동합니다.

return value

없음.

parameters

strId   포커스 줄 객체 아이디를 의미합니다.
editorName  에디터의 id를 의미합니다.

remarks

에디터에서 tab 사용으로 포커스가 빠져나갈 때 특정 객체로 포커스를 이동하고 싶을 때 유용합니다.

sample code

<!-- ..... 생략 ..... -->

<script type="text/javascript" src="raonkeditor/js/raonkeditor.js"></script>

    function setNextTabElementId(){
        // 에디터에서 tab키 사용으로 마지막 포커스를 설정한 ID로 이동합니다.
        RAONKEDITOR.setNextTabElementId('title');
    }

</script>
