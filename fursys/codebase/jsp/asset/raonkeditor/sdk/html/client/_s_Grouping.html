K Editor :: Grouping

XML : <show_toolbar grouping="....."></show_toolbar>, Script : ToolBarGrouping

Script 설정이 우선 적용됩니다.
에디터 아이콘 툴바 그룹핑을 설정합니다.

remarks

기본값은 "1" 이고, "0" 으로 설정 시 아이콘 툴바 그룹핑을 사용하지 않습니다.

노출하고자 하는 아이콘이 많을 경우 아이콘 그룹핑을 사용하시면 유용할 수 있습니다.

sample code

에디터 아이콘 툴바 그룹핑을 사용으로 설정합니다.

XML

.....
<setting>
    .....
    <show_toolbar grouping="1"></show_toolbar>
    .....
</setting>
.....

Script

<script type="text/javascript" src="raonkeditor/js/raonkeditor.js"></script>	
 
<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">    
    <script type="text/javascript">
        var raonkParam = {
            .....
            ToolBarGrouping: '1'
            }
        }

        new RAONKEditor(raonkParam);
    </script>
</div>
