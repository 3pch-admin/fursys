K Editor :: DialogWindow

Script : DialogWindow

에디터 팝업을 Iframe 외부 최 상단 페이지로 설정합니다.

remarks

팝업창, 우클릭 메뉴를 띄우고 싶은 윈도우를 설정합니다.

에디터를 특정 프레임 안에서 생성할 때 프레임 크기가 작은 경우에는
에디터에서 띄우는 팝업창이 가려질 수 있기 때문에
그러한 경우에 설정하여 사용하시면 유용합니다.

sample code

<script type="text/javascript" src="raonkeditor/js/raonkeditor.js"></script>	
 
<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">    
    <script type="text/javascript">

        // 에디터 팝업을 parent.window 으로 설정합니다.
  var raonkParam = {
            .....
            Id: 'editor1',
            DialogWindow: parent.window
        }
        new RAONKEditor(raonkParam);

    </script>       
</div>
