K Editor :: RAONKEDITOR_Resized

void RAONKEDITOR_Resized(editor)

에디터 높이 리사이즈 완료 시 발생합니다.

return value

없음.

parameters

editor  이벤트를 발생시킨 에디터 객체입니다.

remarks

에디터의 설정파일(raonkeditor/config/raonk_editor.xml)의 resizebar 설정값이 1인 경우에 에디터 높이를 리사이즈 할 수 있습니다.
