K Editor :: RAONKEDITOR_BeforePaste

string RAONKEDITOR_BeforePaste(editorID, paramObj)

에디터에 붙여넣기 전 이벤트가 발생합니다.

return value

파라미터로 넘어온 소스를 수정하여 리턴합니다.

parameters

editorID  붙여넣기 이벤트가 발생한 에디터 ID 입니다.
paramObj.strHtml  붙여넣기 된 html 소스 입니다.

remarks

에디터를 띄우는 페이지에 추가한 후 파라미터로 넘어온 소스를 수정하여 리턴해 주면 수정된 소스가 붙여넣기 됩니다.

sample code

<!-- ..... 생략 ..... -->

<script type="text/javascript" src="raonkeditor/js/raonkeditor.js"></script>
<script type="text/javascript">

    function RAONKEDITOR_BeforePaste(editorID, paramObj) {
        // 에디터에 붙여넣기 전 해당 이벤트가 호출됩니다. 
        // 이 부분에서 파라미터를 가공 처리 하신 후 리턴 합니다.

        return paramObj.strHtml;
    }

</script>
