K Editor :: Accessibility

XML : <accessibility>.....</accessibility>, Script : Accessibility
XML : <accessibility validation_item="....."></accessibility>, Script : AccessibilityValidationItems

Script 설정이 우선 적용됩니다.
웹 접근성 단계 설정 및 웹 접근성 검증 항목 중 검증 하고자 하는 항목들을 선택하여 설정합니다.

remarks

웹 접근성 단계는 Accessibility로 설정합니다.
"0" 으로 설정 시 웹 접근성 미설정
"1" 로 설정 시 웹 접근성 위반하면 경고창으로 알려준 후 다음 단계로 진행
"2" 로 설정 시 웹 접근성 위반하면 경고창으로 알려준 후 수정될 때까지 다음단계로 진행하지 않습니다.

웹 접근성 검증 항목은
표 제목 "table_caption", 표 설명 "table_summary", 아이프레임 제목 "iframe_title", 프레임 제목 "frame_title", 이미지 설명 "img_alt", 링크 제목 "a_title", 중복 ID 사용 "duplicate_id" 입니다.

sample code

웹 접근성 단계를 최고단계로 설정합니다.
에디터에 작성된 테이블을 웹 접근성 검증 합니다.

XML

.....
<setting>
    .....
    <accessibility validation_item="table_caption,table_summary">2</accessibility>
    .....
</setting>
.....

<script type="text/javascript" src="raonkeditor/js/raonkeditor.js"></script>	
 
<!-- ..... 생략 ..... -->
