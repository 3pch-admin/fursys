K Editor :: ToolBar1

XML : <tool_bar_1>.....</tool_bar_1>

에디터의 상단(기본) 툴바의 메뉴를 표시하거나 숨기는 기능을 설정합니다.

remarks

상단(기본) 툴바의 메뉴
메뉴명 command 명
새 문서 new
파일로 저장 save
인쇄 print
페이지 분할 pagebreak
실행 취소 undo
다시 실행 redo
복사 copy
잘라내기 cut
붙여넣기 paste
전체선택 select_all
바꾸기 replace
특수문자/기호 symbol
수평선 horizontal_line
하이퍼링크 hyperlink_create
날짜/시간 삽입 insert_datetime
이모티콘 emoticon
문서 배경 이미지 doc_bg_image
이미지 image_create
외부 동영상 공유 insert_link_media
영상 media_create
파일 넣기 insert_file
표 삽입 table_insert
표 선택 table_selection
표 편집 table_group

아래의 표 관련 기능들(아래에 가로줄 삽입, 세로줄 삭제, 오른쪽에 세로줄 삽입, 표 삭제, 셀 합치기, 셀 나누기, 너비를 같게, 높이를 같게)을 하나의 아이콘으로 사용하실수 있습니다.
위에 가로줄 삽입 insert_row_up
아래에 가로줄 삽입 insert_row_down
세로줄 삭제 delete_column
왼쪽에 세로줄 삽입 insert_column_left
오른쪽에 세로줄 삽입 insert_column_right
표 삭제 table_delete
셀 합치기 merge_cell
셀 나누기 split_cell
행 너비 동일하게 table_same_width
열 높이 동일하게 table_same_height
셀 배경색 cell_bg_color
블록 보기 show_blocks
웹 접근성 검증 accessibility_validation
환경설정 setting
전체화면 full_screen
구분선 separator
<remove_item />에 하위 앨리먼트로 메뉴 command 명을 지정해 주면 해당 메뉴를 표시하지 않습니다.
예)<remove_item><item>table_delete</item><item>full_screen</item></remove_item>

sample code
