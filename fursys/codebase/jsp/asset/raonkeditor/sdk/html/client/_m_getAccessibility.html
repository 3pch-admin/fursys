K Editor :: GetAccessibility

string GetAccessibility(editorName)

에디터에 설정된 웹 접근성 단계를 가져옵니다.

return value

0, 1, 2 중 하나의 값을 리턴합니다.

parameters

editorName  에디터의 id를 의미합니다.

remarks

에디터를 1개만 생성했을 경우 parameter를 입력하지 않아도 웹 접근성을 가져올수 있습니다.
예) id가 editor1인 에디터를 생성했을 경우
     RAONKEDITOR.GetAccessibility("editor1"), RAONKEDITOR.GetAccessibility()  모두 editor1 에디터의 웹 접근성 값을 가져옵니다.

에디터를 2개 이상 생성했을 경우 parameter를 반드시 입력해야 합니다.
예) id가 editor1, editor2인 에디터를 생성했을 경우
     RAONKEDITOR.GetAccessibility("editor1")  editor1 에디터의 웹 접근성 값을 가져옵니다.
     RAONKEDITOR.GetAccessibility("editor2")  editor2 에디터의 웹 접근성 값을 가져옵니다.
