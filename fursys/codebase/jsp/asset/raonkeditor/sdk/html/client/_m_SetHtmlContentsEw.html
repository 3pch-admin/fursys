K Editor :: setHtmlContentsEw

void setHtmlContentsEw(value, editor_id)

에디터 생성 전에 호출 하여도 set하려는 소스를 별도로 가지고 있다가 에디터가 생성 완료되면 자동으로 set을 시켜줍니다.

return value

없음.

parameters

value  에디터 영역에 설정할 값을 의미합니다.
editor_id  접근할 에디터의 id를 의미합니다.

remarks

예) id가 editor1인 에디터를 생성했을 경우
    RAONKEDITOR.setHtmlContentsEw(html_source, 'editor1'), RAONKEDITOR.setHtmlContentsEw(html_source)
    모두 editor1 에디터 디자인 영역에 html 소스를 입력합니다.

    에디터를 2개 이상 생성했을 경우 parameter를 반드시 입력해야 합니다.
    예) id가 editor1, editor2인 에디터를 생성했을 경우
    RAONKEDITOR.setHtmlContentsEw(html_source, 'editor1') editor1 에디터 디자인 영역에 html 소스를 입력합니다.
    RAONKEDITOR.setHtmlContentsEw(html_source, 'editor2') editor2 에디터 디자인 영역에 html 소스를 입력합니다.

sample code

<!-- ..... 생략 ..... -->

<script type="text/javascript" src="raonkeditor/js/raonkeditor.js"></script>

        // editor1인 에디터 디자인 영역에 html 소스를 입력합니다.
        RAONKEDITOR.setHtmlContentsEw(html, 'editor1');
    </script>

</div>
