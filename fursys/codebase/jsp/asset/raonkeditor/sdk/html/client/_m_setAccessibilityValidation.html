K Editor :: SetAccessibilityValidation

void SetAccessibilityValidation(editor_id)

웹 접근성 검증 시 위반 항목이 있으면 웹 접근성 검증 팝업 창에 위반 목록을 노출 합니다.

return value

없음.

parameters

editor_id  에디터의 id를 의미합니다.

remarks

웹 접근성 검증 항목은 Table의 Caption과 Summary, Image의 Alt, Iframe의 Title, A의 링크 Title, ID 중복 체크를 검증 합니다.

sample code

<!-- ..... 생략 ..... -->

<script type="text/javascript" src="raonkeditor/js/raonkeditor.js"></script>	
<script type="text/javascript">

    function setAccessibilityValidation() {
        // id가 editor1인 에디터의 웹 접근성 검증 시 위반 항목을 노출 합니다.
        RAONKEDITOR.SetAccessibilityValidation('editor1')
    }

</script>

<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">   
 
    <script type="text/javascript">
         new RAONKEditor({Id:'editor1'});
    </script>           

</div>
