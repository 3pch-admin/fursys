K Editor :: FontFamily

XML : <font_family>.....</font_family>, Script : FontFamilyList

Script 설정이 우선 적용됩니다.
에디터 글꼴 메뉴에 노출할 폰트 리스트를 설정합니다.

remarks

XML:font_family  필요 시 <font>.....</font> 항목을 추가하여 설정합니다.
Script:FontFamilyList  필요 시 FontFamilyList: "FontFamily값"에 콤마(,)로 구분하여 설정합니다.

sample code

에디터 글꼴 메뉴에 노출할 폰트를 맑은 고딕, Tahoma로 설정합니다.

XML

.....
<font_family>
    <font>맑은 고딕</font>
    <font>Tahoma</font>
</font_family>
.....

Script

<script type="text/javascript" src="raonkeditor/js/raonkeditor.js"></script>	
 
<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">  
    <script type="text/javascript">

        var raonkParam = {
            .....
            FontFamilyList: '맑은 고딕,Tahoma'
        }

        new RAONKEditor(raonkParam);
    </script>
</div>
