K Editor :: IsEmpty

bool IsEmpty(editor_id)

에디터 디자인 영역에 내용이 있는지 없는지 체크합니다.

return value

내용이 없으면 true, 있으면 false를 리턴합니다.

parameters

editor_id  내용 유무를 확인할 에디터의 id를 의미합니다.

remarks

에디터를 1개만 생성했을 경우 parameter를 입력하지 않아도 내용 유무를 확인 할 수 있습니다.
예) id가 editor1인 에디터를 생성했을 경우
     RAONKEDITOR.IsEmpty("editor1"), RAONKEDITOR.IsEmpty()  모두 editor1 에디터의 디자인 영역 내용 유무를 확인합니다.

에디터를 2개 이상 생성했을 경우 parameter를 반드시 입력해야 합니다.
예) id가 editor1, editor2인 에디터를 생성했을 경우
     RAONKEDITOR.IsEmpty("editor1")  editor1 에디터의 디자인 영역 내용 유무를 확인합니다.
     RAONKEDITOR.IsEmpty("editor2")  editor2 에디터의 디자인 영역 내용 유무를 확인합니다.

게시판 등에서 에디터에 사용자가 내용을 입력했는지 확인할 때 유용하게 사용할 수 있습니다.
