K Editor :: InitServerXml

Script : InitServerXml

만일 에디터 생성시 로드할 XML에 파일 저장 경로등과 같은 값의 노출이 문제가 되는 경우
XML파일을 암호화 하여 처리 하는 방식을 의미합니다.

remarks

기본값은 빈값으로 기본 XML을 읽는 것 입니다. ("raonkeditor/config/raonkeditor.config.xml")

sample code

Client

<script type="text/javascript" src="raonkeditor/js/raonkeditor.js"></script>	
<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        // Full URL(http 포함)로 다음과 같이 설정합니다.
        // Full URL은 K Editor Handler경로로 지정하며, f 파라미터의 값은 서버에서 처리할 XML 파일명을 지정합니다.
        var raonkParam = {
            .....
            InitServerXml: 'http://localhost/raonkeditor/handler/raonkhandler.jsp?f=raonkeditor.config.xml'
        }

        new RAONKEditor(raonkParam);
    </script>
</div>

Server

.NET

editor.settingVo.SetConfigPhysicalPath("D:\\config"); // 환경설정파일(xml)이 존재하는 서버의 폴더 경로

JAVA

editor.settingVo.setConfigPhysicalPath("D:\\config"); // 환경설정파일(xml)이 존재하는 서버의 폴더 경로
