K Editor :: SetUserCssText

void SetUserCssText(cssText, editorName)

에디터에 사용자의 css를 style tag로 적용합니다.

return value

없음.

parameters

cssText  사용자가 적용할 css 값 입니다.
editorName  접근할 에디터의 id 입니다.
