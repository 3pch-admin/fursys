K Editor :: AutoDestroy

XML : <auto_destroy>.....</auto_destroy>, Script : AutoDestroy.Use
XML : <auto_destroy move_cursor="....."></auto_destroy>, Script : AutoDestroy.MoveCursor

XML:auto_destroy,Script:AutoDestroy.Use  에디터 언로드 시 메모리 해제 여부를 설정합니다.
XML:move_cursor,Script:AutoDestroy.MoveCursor  에디터 destroy시 커서를 다른 객체로 옮겨주도록 설정합니다.

remarks

XML:auto_destroy,Script:AutoDestroy.Use  기본값은 0이고, 1로 설정하면 에디터 언로드 시 destroy합니다.
XML:move_cursor,Script:AutoDestroy.MoveCursor  기본값은 0이고, 1로 설정하면 destroy 시 커서를 다른 객체로 옮겨줍니다.

sample code

에디터 언로드 시 destroy 사용으로 설정, destroy 시 커서를 다른 객체로 옮김으로 설정합니다.
