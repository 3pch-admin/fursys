K Editor :: ShowTopMenu

void ShowTopMenu(value, editor_id)

에디터의 메뉴 바를 표시하거나 숨기는 기능을 설정합니다.

return value

없음.

parameters

value  에디터 영역에 설정할 값을 의미합니다.
editor_id  에디터의 id를 의미합니다.

remarks

사용 용도에 따라서 메뉴 바를 표시하지 않기를 원할 때 사용하면 유용 할 수 있습니다.
"0" 이면 메뉴 바를 숨기고, "1" 이면 메뉴 바를 표시합니다.

sample code

<!-- ..... 생략 ..... -->

<script type="text/javascript" src="raonkeditor/js/raonkeditor.js"></script>	
<script type="text/javascript">
function showTopMenu(){
    // 에디터의 메뉴 바를 보이지 않도록 합니다.
    RAONKEDITOR.ShowTopMenu(0, 'editor1');
}

</script>
