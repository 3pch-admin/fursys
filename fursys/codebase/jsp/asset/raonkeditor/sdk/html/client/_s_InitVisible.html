K Editor :: InitVisible

Script : InitVisible

에디터를 화면에 표시할지 여부를 설정합니다.

remarks

"true"로 설정 시 에디터를 화면에 표시하고, "false"로 설정 시 표시하지 않습니다.
기본값은 "true" 입니다.

sample code

<script type="text/javascript" src="raonkeditor/js/raonkeditor.js"></script>	

<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:400px">  
    <script type="text/javascript">
        // editor1 에디터는 화면에 표시가 됩니다.
        var raonkParam1 = {
            Id: 'editor1',
            .....
            InitVisible: true
        }
        
        new RAONKEditor(raonkParam1);
    </script>

    <script type="text/javascript">
        // editor2 에디터는 화면에 표시되지 않습니다.
        var raonkParam2 = {
            Id: 'editor2',
            .....
            InitVisible: false
        }
        
        new RAONKEditor(raonkParam2);
    </script>
</div>
