K Editor :: SetRulerPosition

void SetRulerPosition(position, editor_id)

에디터 가로 눈금자 포인터의 위치 값을 설정합니다.

return value

없음.

parameters

position  눈금자 포인터의 위치 값을 의미합니다.
editor_id  에디터의 id를 의미합니다.

remarks

기본값은 "680" 입니다.
setRulerPosition 을 설정하기 위해서는 에디터의 설정파일(raonkeditor/config/raonkeditor.xml)의
use_ruler 을 "1" 로 설정해 주어야 합니다.

sample code

<!-- ..... 생략 ..... -->

<script type="text/javascript" src="raonkeditor/js/raonkeditor.js"></script>	
<script type="text/javascript">

    // 에디터 가로 눈금자 포인터의 위치 값을 700으로 설정합니다.
    function setRulerPosition(){
        RAONKEDITOR.SetRulerPosition("700", 'editor1');
    }

</script>	

<!-- ..... 생략 ..... -->

    <!-- 에디터 생성 -->  
    <script type="text/javascript">
         new RAONKEditor({Id:'editor1'});
    </script>

</div>
