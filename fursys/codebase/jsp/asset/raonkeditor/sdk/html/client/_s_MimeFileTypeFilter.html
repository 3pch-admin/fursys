K Editor :: MimeFileTypeFilter

Script : MimeFileTypeFilter

MIME 인코딩 대상 파일을 설정합니다.

remarks

인코딩 대상파일 설정값은
"1"로 설정 시 "Image", "2"로 설정 시 "css", "4"로 설정 시 "js", "8"로 설정 시 "동영상, swf 등 Object 파일" 입니다.
기본값은 "1"로 "Image" 입니다.

예) 이미지와 js 파일 종류 모두를 인코딩 대상으로 지정하기 위해서는
     "Image"의 설정값인 "1"과 "js"의 설정값인 "4"를 더한 "5" 로 설정합니다.

sample code

<script type="text/javascript" src="raonkeditor/js/raonkeditor.js"></script>	
 
<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">  
    <script type="text/javascript">

        var raonkParam = {
            .....
            // 이미지와 js 파일 종류 모두를 인코딩 대상으로 설정합니다.
            MimeFileTypeFilter: '5'
        }

        new RAONKEditor(raonkParam);
    </script>
</div>
