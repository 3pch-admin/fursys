K Editor :: PasteToImage

XML : <paste_to_image>.....</paste_to_image>, Script : PasteToImage

Script 설정이 우선 적용됩니다.
에디터 마우스 우클릭 시 이미지로 붙여넣기 항목을 설정합니다.

remarks

기본값은 "1" 이고 기본 붙여넣기에 이미지로 붙여넣기를 설정, "0" 으로 설정 시 우클릭 이미지로 붙여넣기 항목을 설정합니다.

sample code


에디터 마우스 우클릭 시 이미지로 붙여넣기 항목으로 설정합니다.

.....
<setting>
    .....
    <paste_to_image>0</paste_to_image>
    .....
