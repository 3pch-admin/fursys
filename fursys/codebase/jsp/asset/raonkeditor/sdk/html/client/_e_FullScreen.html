K Editor :: RAONKEDITOR_FullScreen

void RAONKEDITOR_FullScreen(editorId, editorFrameStatus)

에디터 화면전환(전체화면, 기본화면) 시 발생합니다.

return value

없음.

parameters

editorId  화면전환된 에디터의 id입니다.
editorFrameStatus  전환된 화면 상태값 입니다.

remarks

editorFrameStatus 값은 전체화면 "full_size", 기본화면 빈값("")입니다.
