K Editor :: ShowStatusbar

void ShowStatusbar(value, editor_id)

에디터의 상태 바는 "디자인", "HTML" 및 "미리 보기", "TEXT" 탭으로 구성되어 있습니다.
이 설정은 상태 바를 표시하거나 숨기는 기능을 설정합니다.

return value

없음.

parameters

value  상태바에 대한 설정할 값을 의미합니다.
editor_id  접근할 에디터의 id를 의미합니다.

remarks

사용 용도에 따라서 하단 상태 바를 표시하지 않기를 원할 때 사용하면 유용 할 수 있습니다.
"0" 이면 상태 바를 숨기고, "1" 이면 상태 바를 표시합니다.

sample code

<!-- ..... 생략 ..... -->

<script type="text/javascript" src="raonkeditor/js/raonkeditor.js"></script>	
<script type="text/javascript">

    function showStatusbar(){
        // 에디터의 상태바를 숨기는 기능으로 설정합니다.
        RAONKEDITOR.ShowStatusbar(0, 'editor1');
    }

</script>
