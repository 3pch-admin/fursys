K Editor :: Hidden

void Hidden(editor_id)

에디터를 화면에서 숨깁니다.

return value

없음.

parameters

editor_id  숨길 에디터의 id를 의미합니다.

remarks

에디터를 1개만 생성했을 경우 parameter를 입력하지 않아도 에디터를 숨길 수 있습니다.
예) id가 editor1인 에디터를 생성했을 경우
     RAONKEDITOR.Hidden("editor1") editor1 에디터를 화면에서 숨김니다.

에디터를 2개 이상 생성했을 경우 parameter를 반드시 입력해야 합니다
예) id가 editor1, editor2인 에디터를 생성했을 경우
     RAONKEDITOR.Hidden("editor1")  editor1 에디터를 화면에서 숨김니다.
     RAONKEDITOR.Hidden("editor2")  editor2 에디터를 화면에서 숨김니다.
