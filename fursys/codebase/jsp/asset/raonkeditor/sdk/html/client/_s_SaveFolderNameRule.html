K Editor :: SaveFolderNameRule

XML : <save_foldername_rule>.....</save_foldername_rule>, Script : SaveFolderNameRule

Script 설정이 우선 적용됩니다.
ToSavePathUrl에 설정된 폴더 하위들의 저장 체계를 설정합니다.

remarks

기본값은 "YYYY/MM/" 입니다.

설정 방법은
사용하지 않음 => "/",
년도 => "YYYY/",
년도/월 => "YYYY/MM/",
년도/월/일 => "YYYY/MM/DD/" 중 하나의 값을 사용합니다.

폴더가 존재하지 않으면 자동으로 생성되며, 서버의 날짜가 기준이 됩니다.

sample code

에디터 폴더들에 대한 저장 체계를 년도/월/일 로 설정합니다.

XML

.....
<uploader_setting>
    .....
    <save_foldername_rule>YYYY/MM/DD/</save_foldername_rule>
    .....
</uploader_setting>
.....

Script

<script type="text/javascript" src="raonkeditor/js/raonkeditor.js"></script>
