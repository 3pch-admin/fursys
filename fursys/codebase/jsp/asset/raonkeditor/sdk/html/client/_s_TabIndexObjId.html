K Editor :: TabIndexObjId

Script : TabIndexObjId

에디터 로드 후 div 같은 객체에 포커스를 주고 싶을 때 tabindex를 0으로 설정할 id값을 받습니다.

remarks

필요 시 TabIndexObjId: "아이디값" 으로 설정합니다.

sample code

<script type="text/javascript" src="raonkeditor/js/raonkeditor.js"></script>	
 
<!-- ..... 생략 ..... -->
 
<div border="0" style="width:900px;height:550px">    
    <script type="text/javascript">

        // 에디터 div 객체에 id값 title 으로 설정합니다.
        var raonkParam = {
            .....
            TabIndexObjId: 'title'
        }

        new RAONKEditor(raonkParam);


    </script>       
</div>
