K Editor :: RemoveItem

XML : <remove_item>.....</remove_item>, Script : RemoveItem

Script 설정이 우선 적용됩니다.
에디터에서 미표시 하려는 command를 설정합니다.

remarks

XML:remove_item  필요 시 <item>.....</item> 항목을 추가하여 설정합니다.
Script:RemoveItem  필요 시 RemoveItem: "command명"에 콤마(,)로 구분하여 설정합니다.

sample code

메뉴 중 새 문서, 파일열기를 미표시로 설정합니다.

XML

.....
<remove_item>
    <item>new</item>
    <item>file_open</item>
</remove_item>
.....

Script

<script type="text/javascript" src="raonkeditor/js/raonkeditor.js"></script>
