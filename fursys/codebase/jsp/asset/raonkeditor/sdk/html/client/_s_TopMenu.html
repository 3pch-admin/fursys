m_file  메뉴바에서 [파일] 메뉴를 표시하거나 숨기는 기능을 설정합니다. 하위메뉴는 다음과 같습니다.

메뉴바에서 [파일] 메뉴의 하위 메뉴
메뉴명 command 명
새 문서 new
파일열기 file_open
템플릿 template
레이아웃 layout
파일로 저장 save
자동저장내용 불러오기 load_auto_save
인쇄 print
페이지 분할 pagebreak
<remove_item />에 하위 앨리먼트로 하위 메뉴 command 명을 지정해 주면 해당 하위 메뉴를 표시하지 않습니다.
예)<remove_item><item>new</item><item>print</item></remove_item>
