K Editor :: ImageBaseUrl

XML : <image_base_url>.....</image_base_url>, Script : ImageBaseUrl

Script 설정이 우선 적용됩니다.
에디터 이미지 기본 주소를 설정합니다.

remarks

url은 http://를 포함한 전체 경로를 입력해야 합니다.
예) http://localhost/raonkeditor/
