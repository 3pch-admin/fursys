K Editor :: GetHtmlContents

void GetHtmlContents(paramObj, editor_id)

에디터에서 작성한 내용을 가져 올 수 있습니다.

return value

없음.

parameters

paramObj.type  에디터에서 가져올 소스를 설정합니다.
paramObj.type : text   에디터 디자인 영역의 HTML 소스에서 <body> 태그 내부의 텍스트만 가져옵니다.
paramObj.type : htmlexwithdoctype   에디터 디자인 영역의 HTML 소스(<DOCTYPE> 태그 포함)를 리턴합니다.
paramObj.type : htmlex   에디터 디자인 영역의 HTML 소스에서 <html> 부터 </html>까지의 소스를 가져옵니다.
paramObj.type : html  에디터 디자인 영역의 HTML 소스에서 <html> 태그 내부의 소스를 가져옵니다.
paramObj.type : bodyex   에디터 디자인 영역의 HTML 소스에서 <body> 부터 </body>까지의 소스를 가져옵니다.
paramObj.type : body  
paramObj.callback  가져온 결과값을 받기 위한 콜백 함수입니다.
paramObj.strEditorId  소스를 가져온 에디터의 ID입니다.
paramObj.strData  에디터에서 가져온 HTML 소스입니다.
editor_id  접근할 에디터의 id를 의미합니다.

remarks

GetHtmlContents API는 아래와 같이 get 형태와 callback 함수를 지정해 주시면 됩니다.
예: GetHtmlContents({type: 'text', callback: function (paramObj) { alert(paramObj.strData); }}, editorId)

예) id가 editor1인 에디터를 생성했을 경우
RAONKEDITOR.SetHtmlContents(paramObj, "editor1"), RAONKEDITOR.SetHtmlContents(html_source)
모두 editor1 에디터 디자인 영역의 HTML 소스에서 <body> 태그 내부의 텍스트를 경고창으로 띄웁니다.

에디터를 2개 이상 생성했을 경우 parameter를 반드시 입력해야 합니다.
예) id가 editor1, editor2인 에디터를 생성했을 경우
RAONKEDITOR.GetHtmlContents(paramObj, 'editor1') editor1 에디터 디자인 영역의 HTML 소스에서 <body> 태그 내부의 텍스트를 경고창으로 띄웁니다.
RAONKEDITOR.GetHtmlContents(paramObj, 'editor2') editor2 에디터 디자인 영역의 HTML 소스에서 <body> 태그 내부의 텍스트를 경고창으로 띄웁니다.

            var msg = paramObj.strEditorId + ' 작성 내용 : \n';

            alert(msg);
        };

        RAONKEDITOR.GetHtmlContents({
            type: 'htmlex',
            callback: fn_callback
        }, editorID);
    }
