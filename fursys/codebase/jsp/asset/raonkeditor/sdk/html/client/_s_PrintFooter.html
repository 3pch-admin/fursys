K Editor :: PrintFooter

XML : <print_footer_left>.....</print_footer_left>
XML : <print_footer_center>.....</print_footer_center>
XML : <print_footer_right>.....</print_footer_right>

인쇄 시 Footer 영역의 내용을 설정합니다.

remarks

print_footer설명
&w : 현재 페이지의 title 값
&u : 현재 페이지의 url
&p : 현재 페이지 번호
&P : 전체 페이지 수
&d : 약식 날짜 예) 2017-00-00
&D : 긴 형식 날짜 예) 2017년 00월 00일 O요일
&t : 시간 예) 오후 1:40:00
&T : 24시간 형식 시간 예) 오후 13:40:00
이외의 값은 넣은 값 그대로 들어감.
