K Editor :: SetDefaultStyle

XML : <set_default_style>.....</set_default_style>
XML : <set_default_style set_style="....."></set_default_style>
XML : <set_default_style body_id=".....""></set_default_style>

XML:set_default_style  에디터 로드 시 기본 스타일이 없을 경우 body 영역에 에디터 스타일을 설정합니다.
XML:set_style  기본 스타일 사용으로 설정 시 RAONKEditor 기본 CSS 로드 사용 여부를 설정합니다.
XML:body_id  에디터 body 영역에 id 값을 설정합니다.

remarks

XML:set_default_style  기본값은 "0" 이고 에디터 기본 스타일 사용,
"1"로 설정 시 에디터 body영역만 스타일 설정,
"2"로 설정 시 getAPI 사용으로 태그에 inline 스타일 설정됩니다.
XML:set_style  기본값은 "1" 이며 스타일 적용, "0" 이면 스타일을 미적용 합니다.
set_default_style 설정값이 "0" 이고,
set_style 설정값을 "0" 으로 설정하면 외부 URL 로드 시 RAONKEditor 기본 CSS를 로드하지 않습니다.
XML:body_id  기본값은 "keditor_body"이고, "사용자 설정값"으로 설정합니다.

sample code

에디터 로드 시 기본 스타일이 없을 경우 body 영역에 에디터 스타일을 getAPI 사용으로 태그에 inline 스타일 설정합니다.
기본 스타일 사용으로 설정 시 RAONKEditor 기본 CSS 로드 사용 여부를 사용 안함으로 설정합니다.
에디터 body 영역에 id 값을 "custom_body_id"로 설정합니다.

.....
<setting>
    .....
    <set_default_style set_style="0" body_id="custom_body_id">2</set_default_style>
    .....
</setting>
.....
