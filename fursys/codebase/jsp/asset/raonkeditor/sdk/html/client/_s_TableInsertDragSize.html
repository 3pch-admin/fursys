K Editor :: TableInsertDragSize

XML : <insert_drag_size></insert_drag_size>
Script : TableInsertDragSize

Script 설정이 우선 적용됩니다.
표 삽입 메뉴의 최대 선택 개수를 설정합니다.

remarks

기본값은 "10,10" 이고, "행,열" 개수로 설정합니다.

sample code

표 삽입 메뉴의 최대 선택 개수를 15행,10열으로 설정합니다.

XML

.....
<setting>
    .....
    <table_property>
        .....
        <insert_drag_size>15,10</insert_drag_size>
        .....
    </table_property>
    .....
</setting>
.....

Script

<script type="text/javascript" src="raonkeditor/js/raonkeditor.js"></script>	
 
<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">  
    <script type="text/javascript">
        var raonkParam = {
            .....
            TableInsertDragSize: '15,10'
        }

        new RAONKEditor(raonkParam);
    </script>
</div>
