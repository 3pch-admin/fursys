K Editor :: EditorHolder

Script : EditorHolder

에디터를 특정 영역에 생성할 수 있도록 합니다.

remarks

에디터를 생성할 영역의 id값을 입력합니다.
필요시 EditorHolder의 값을 "아이디값" 으로 설정합니다.

sample code

<script type="text/javascript" src="raonkeditor/js/raonkeditor.js"></script>	

<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        // 에디터에 생성할 객체의 ID를 설정합니다.
        var raonkParam = {
            .....
            EditorHolder: 'editor_area'
        }

        new RAONKEditor(raonkParam);
    </script>
</div>
