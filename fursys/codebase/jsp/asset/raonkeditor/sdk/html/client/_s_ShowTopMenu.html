K Editor :: ShowTopMenu

XML : <show_topmenu>.....</show_topmenu>, Script : ShowTopMenu

Script 설정이 우선 적용됩니다.
에디터 메뉴바를 보이거나 숨기는 기능을 설정합니다.
사용 용도에 따라서 메뉴바를 숨김으로 설정하기를 원할 때 사용하면 유용 할 수 있습니다.

remarks

기본값은 "1" 이고 메뉴바 보임, "0" 으로 설정 시 숨김으로 설정합니다.

sample code

            
에디터의 메뉴바를 숨김으로 설정합니다.

XML

.....
<setting>
    .....
    <show_topmenu>0</show_topmenu>
    .....
</setting>
.....

Script

<script type="text/javascript" src="raonkeditor/js/raonkeditor.js"></script>	
 
<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">    
    <script type="text/javascript">
        var raonkParam = {
            .....
            ShowTopMenu: '0'
        }

        new RAONKEditor(raonkParam);
    </script>
</div>
