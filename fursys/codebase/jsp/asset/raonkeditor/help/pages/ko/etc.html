한글 입력 문제(Firefox 41.0버전)

현상

Firefox 41.0 버전에서 K Editor에 한글 입력 시 줄 바꿈 후 텍스트를 입력하기 시작하면 커서가 이전 줄로 이동, 일정 시간 후 커서가 텍스트 맨 첫 줄로 강제 이동 등이 모든 HTML 에서 발생 하고 있습니다.

원인

Firefox 에서 한글 입력 문제는 41.0버전에 새롭게 추가된 윈도비스타 이후 OS의 TSF를 사용한 향상된 IME 지원 기능인 Text Services Framework이 원인으로 밝혀져 있습니다.

해결 방법

Firefox 42.0 버전으로 업그레이드를 하시면 됩니다. (자동 업그레이드 지원 안함)


특수문자, 한자 입력 문제(Chrome 77.0.3865.90버전)

현상

Chrome 77.0.3865.90 이후 버전에서 K Editor에 특수문자, 한자 입력 후 한글 입력 시 특수문자, 한자가 사라지는 현상이 모든 HTML에서 발생하고 있습니다.

원인

Chrome 브라우저 자체 버그

해결 방법

Chrome 최신 버전(81.0.4044.113 이후 버전)으로 업그레이드를 하시면 됩니다. (자동 업그레이드 지원)
